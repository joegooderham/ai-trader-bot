# dashboard/frontend/src/

React application source code.

| Path | Purpose |
|------|---------|
| `App.jsx` | Root component with React Router — defines all 14 routes |
| `main.jsx` | Entry point — renders App inside BrowserRouter |
| `index.css` | Tailwind CSS directives and global styles |
| `pages/` | Page-level components (one per route) |
| `components/` | Reusable UI components shared across pages |
| `hooks/` | Custom React hooks for data fetching and commands |
